# EmpowerHer Assets

This folder contains images and icons for the EmpowerHer website.

## Recommended Image Sources
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com

## Jasmin Flower Images
The website uses Jasmin flower aesthetic. Recommended images:
- Jasmin flower photos for hero sections
- Flower icons for UI elements
- Background patterns with floral themes

## Usage
Place all images in this folder and reference them in HTML as:
```html
<img src="{{ url_for('static', filename='assets/images/your-image.jpg') }}">
```

## Note
For a production website, download and add actual images from the recommended sources.
